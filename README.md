# Wolf:ET Pack

A [Wolfenstein: Enemy Territory](https://www.splashdamage.com/games/wolfenstein-enemy-territory/) sound pack for [OpenPeon](https://openpeon.com).

## Installation

Install via OpenPeon:

```
openpeon install wolf-et-pack
```

## Categories

| Category | Sounds | Description |
|---|---|---|
| `session.start` | 17 | Map briefings and round start cues |
| `task.acknowledge` | 46 | Dynamite planted, objectives taken, airstrikes called |
| `task.complete` | 59 | Objectives destroyed/secured, promotions |
| `task.error` | 32 | Construction failed, airstrikes denied/aborted, objectives lost |
| `input.required` | 20 | Need engineer, need medic, need field ops |
| `resource.limit` | 12 | Time warnings, mines spotted |

Sounds are available in both Allied and Axis voice variants.
